SMSCenter
=========

Библиотека-обёртка для SMSCenter API

Примеры использования:
```php
<?php
// Инициализация
$sms = new SMSCenter(array(
	'login'	   => 'ivan',
	'password' => md5('ivanovich'),
	'charset' => SMSCenter::CHARSET_UTF8
));

// Отправка сообщения
$sms->sendMessage('+7991111111', 'Превед, медведы!', 'SuperIvan');

// Отправка сообщения на 2 номера
$sms->send(array('+7(999)1111111', '+7(999)222-22-22'), 'Превед, медведы! Одно сообщение на 2 номера.', 'SuperIvan');

// Получение баланса
$sms->getBalance();

// Получение информации об операторе
$sms->getOperatorInfo('7991111111');

// Получения статуса сообщения
$sms->getStatus('+7991111111', 6, SMSCenter::STATUS_INFO_EXT);

// Получение стоимости рассылки
$sms->getCost('+7991111111', 'Начало около 251 млн лет, конец — 201 млн лет назад, длительность около 50 млн лет.');
?>
```
